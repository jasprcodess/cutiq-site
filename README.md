Hey 👋 As we're in a private repo, no need to be overly formal. First of all, thank you for purchasing AppView and supporting my work!

Just to reassure you, this is not a project that was thrown together in a weekend. It came from my personal need when I was rebuilding a website for my app Ripples. I wasted so much time overthinking every little detail, and I knew that soon I would need to do it all over for my second app. So I've decided to pack everything into a reusable and customizable template. It tuned out to be way more work than I anticipated (story of my life tbh), but I'm happy with the final result and truly think it will save you a lot of time and headaches.

If case you have feedback, questions, or need help with anything, email is the fastest way to get the reply: <a href="mailto:mykola@appview.dev">mykola@appview.dev</a>.

For bugs and feature requests, please create an issue in this repo.
